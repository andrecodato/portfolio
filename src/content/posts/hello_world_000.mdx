Este é o primeiro post do blog, escrito em **MDX**. Isso significa que além de Markdown normal, eu posso usar componentes React diretamente no texto.

## O que é MDX?

MDX é uma combinação de **Markdown** com **JSX**. Você escreve conteúdo como faria em qualquer arquivo `.md`, mas com superpoderes:

- Importar e usar componentes React
- Escrever expressões JavaScript inline
- Estilizar com Tailwind diretamente nos componentes

## Exemplo de código

```typescript
const greeting = "Olá do blog!";
console.log(greeting);
```

## Por que MDX?

> Porque escrever conteúdo deveria ser tão flexível quanto escrever código.

Com MDX posso manter os posts versionados no repositório, sem precisar de um CMS externo. Simples, rápido e com controle total.
