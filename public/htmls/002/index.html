O que é HTML?

O HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar sites.

O que é CSS?

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para fazer a parte visual de uma estrutura HTML.

Para que serve a tag head?

A tag head não é visível ao usuário e serve para definir metadados como Título, Autor, Charset, Links para arquivos CSS e scripts JavaScript.

Para que serve a tag meta?

A tag meta define um metadado, pode ser charset, name, content, entre outros.

O que é charset? Qual é o charset utilizado no Brasil?

O charset é o conjunto de caracteres que podemos utilizar no documento, cada linguagem usa um padrão, aqui no Brasil, utilizamos o UTF-8.

Como funciona a tag img?

Utilizamos a tag img para exibir imagens.
Nela podemos definir atributos como src (Caminho da imagem), alt (Texto alternativo. Muito utilizado para questões de acessibilidade!), entre outros.

Como funciona a tag a?

Utilizamos a tag a quando queremos fazer um Hyperlink, ou seja, queremos criar um link para outra página ou site.
Nela podemos definir href (Caminho do link), target(Onde o link será aberto), entre outros.
Exemplo: <a href="https://www.google.com" target="_blank">Google</a>
Resultando em Google

Como funciona a tag h1? e a h6?

As tags h1 a h6 são utilizadas para definir títulos, sendo o h1 o mais importante e o h6 o menos importante.
Exemplos: <h1>Título 1</h1>
<h6>Título 6</h6>
Resultando em

Título 1

e
Título 6

Como usar o atributo style nas tags?

O atributo style é utilizado para aplicar estilos diretamente em uma tag HTML.
Por exemplo: <p style="font-size: 50px;">Texto grande</p>
Resultando em

Texto grande

Qual a propriedade utilizada no css para alterar a cor da letra em um texto? Faça um exemplo.

A propriedade utilizada no CSS para alterar a cor da letra em um texto é color.
Por exemplo: <p style="color: blue;">Texto azul</p>
Resultando em

Texto azul
